Notes:

Template Literals: use backticks. ${} to add expressions. Template Literals are "Super-powered" strings.

To know if a variable holds a number, you can use number.isFinite(x);. This will return true or false.

Javascript uses "Data Coercion", but it's better to be explicit on the datatypes to avoid "funny" behaviors like '2' + 4 = '24'

While undefined means that the variable hasn't been assigned yet, null means "no value". The difference is that null is coerced into 0 when used in an arithmetic operation.

Ternary Operators are a shorthand for writing if...else loops. The syntax is: condition ? (code to run if true) : (code to run if false);

Loops. Js has While loops, Do...While loops, and For loops

Working with arrays, instead of a for loop, we can use a for-of loop:

for(const value of arrayName){
actions;
}

There are three ways to declare a function: with a function litera, a function expression, and function constructor.

DRY - Don't Repeat Yourself.
